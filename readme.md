## Clone

Изначально нужно склонить проект к себе

```bash
git clone https://github.com/firsov-dmitriy/whatsAppTest.git
```

## Установка зависимостей

Use the package manager npm

```bash
yarn
```

or

```bash
npm i
```

## Environments

```bash
API_URL - link to api service (https://api.green-api.com)
PORT - default (3000)
```

## Usage

```bash
yarn start dev
```

or

```bash
npm start dev
```
